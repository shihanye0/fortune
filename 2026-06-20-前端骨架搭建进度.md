## 目录结构

```
client/src/
├── app/                          # 应用入口
│   ├── App.vue                   # 根组件
│   ├── main.ts                   # 入口文件
│   ├── router.ts                 # 路由配置
│   └── styles/global.css         # 全局样式
├── features/                     # 功能模块
│   ├── home/views/HomeView.vue   # 首页
│   ├── auth/                     # 认证模块
│   │   ├── views/LoginView.vue   # 登录页
│   │   ├── views/RegisterView.vue # 注册页
│   │   ├── api/auth-api.ts       # API 调用
│   │   ├── stores/auth.ts        # 状态管理
│   │   └── types/user.ts         # 类型定义
│   ├── fortune/views/            # 运势模块（占位）
│   ├── divination/views/         # 占卜模块（占位）
│   └── profile/views/            # 个人中心（占位）
├── shared/                       # 共享资源
│   ├── api/client.ts             # Axios 客户端
│   ├── components/AppLayout.vue  # 全局布局
│   └── types/api.ts              # API 类型
└── test/setup.ts                 # 测试配置
```
