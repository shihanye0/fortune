## 技术栈

### 后端
- Python 3.10 + FastAPI + SQLAlchemy
- SQLite（开发）/ MySQL（生产）
- JWT 认证 + bcrypt 密码哈希
- Alembic 数据库迁移

### 前端
- Vue 3 + TypeScript + Vite 6
- Element Plus UI 组件库
- Pinia 状态管理
- Vue Router 路由
- Axios HTTP 客户端
- Vitest 测试框架

### CI/CD
- GitHub Actions 每日推送（每小时触发）
- 自动运行前后端测试
