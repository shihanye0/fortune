# Spec 011: 每日运势推算

> 模块：命理计算引擎 | 优先级：P0 | 预计：1 天

## 功能描述

综合用户八字和当日流日干支，推算事业、财运、感情、健康四个维度的运势。

## 输入/输出

**输入：** 用户八字档案 + 当日日期

**输出：**
```json
{
  "heavenly_stem": "甲",
  "earthly_branch": "子",
  "overall_score": 4,
  "career_fortune": {"score": 4, "detail": "..."},
  "wealth_fortune": {"score": 3, "detail": "..."},
  "love_fortune": {"score": 5, "detail": "..."},
  "health_fortune": {"score": 3, "detail": "..."},
  "lucky_color": "红色",
  "lucky_number": "3, 8",
  "lucky_direction": "东方"
}
```

## 验收标准

```
Scenario: 每日运势推算正确
  Given 用户八字为 "庚午 辛巳 丙寅 壬辰"
  And   今日干支为 "甲子"
  When  调用每日运势推算
  Then  应返回四个维度的评分（1-5）
  And   应返回幸运色、数字、方位
  And   评分应基于日主与流日的五行生克关系
```

## 技术要点

- 评分基于日主与流日天干的十神关系
- 综合考虑流日地支与八字地支的刑冲合害
- 幸运色/数字/方位基于五行喜用神推算

## 依赖项

- 前置：010-八字排盘
