# Spec 021: 运势详情与反馈

> 模块：运势查看 | 优先级：P0 | 预计：1 天

## 功能描述

用户查看运势详情，并对运势进行反馈（评分、标签、文字）。

## API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/v1/fortunes/:id | 运势详情 |
| POST | /api/v1/fortunes/:id/feedback | 提交反馈 |

**反馈请求体：**
```json
{
  "rating": 4,
  "tags": ["准", "有帮助"],
  "feedback_text": "财运分析很准，希望更详细"
}
```

## 验收标准

```
Scenario: 查看运势详情
  Given 运势ID=1 存在
  When  查看详情
  Then  应返回完整运势内容

Scenario: 提交反馈
  Given 运势ID=1 存在
  When  用户提交评分 4 星和文字反馈
  Then  反馈应保存成功

Scenario: 修改已有反馈
  Given 用户昨天给运势打了 3 星
  When  用户修改为 5 星
  Then  反馈应更新成功
```

## 技术要点

- 反馈允许修改（PUT 语义）
- 反馈标签预定义：准/不准/有帮助/需改进
