# Spec 025: 首页与导航

> 模块：前端页面 | 优先级：P0 | 预计：0.5 天

## 功能描述

产品首页展示和全局导航布局。

## 页面结构

```
┌─────────────────────────────────┐
│  Logo    [运势] [占卜] [个人]   │  ← 顶栏导航
├─────────────────────────────────┤
│                                 │
│     🌟 命理运势系统              │  ← 首页内容
│     每日精准运势，尽在掌握       │
│                                 │
│     [立即注册]  [了解更多]       │
│                                 │
└─────────────────────────────────┘
```

## 验收标准

```
Scenario: 首页展示
  Given 用户访问首页
  Then  应显示产品介绍
  And   应显示注册入口

Scenario: 导航栏功能
  Given 用户已登录
  When  点击导航栏 "运势"
  Then  应跳转到运势页面
```
