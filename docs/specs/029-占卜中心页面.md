# Spec 029: 占卜中心页面

> 模块：前端页面 | 优先级：P1 | 预计：1 天

## 功能描述

占卜中心页面，支持六爻占卜和奇门遁甲。

## 页面结构

```
┌─────────────────────────────────┐
│  占卜中心                       │
├─────────────────────────────────┤
│  选择占卜方式：                  │
│  [六爻占卜]  [奇门遁甲]         │
│                                 │
│  输入问题（选填）：              │
│  [________________________]     │
│                                 │
│  [起卦/起盘]                    │
│                                 │
│  ── 结果 ──                     │
│  卦象/盘面展示                   │
│  LLM 解读文字                   │
│                                 │
│  [反馈]                         │
└─────────────────────────────────┘
```

## 验收标准

```
Scenario: 六爻占卜
  Given 用户输入问题
  When  选择六爻占卜并点击起卦
  Then  应显示卦象和解读

Scenario: 奇门遁甲实时看盘
  Given 用户选择实时看盘
  When  点击起盘
  Then  应显示当前时辰盘面和解读
```

## 依赖项

- 前置：022-六爻占卜交互、023-奇门遁甲交互
