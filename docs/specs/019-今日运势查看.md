# Spec 019: 今日运势查看

> 模块：运势查看 | 优先级：P0 | 预计：0.5 天

## 功能描述

用户查看今日运势，如果未生成则显示提示。

## API

```
GET /api/v1/fortunes/today
```

**响应：**
```json
{
  "success": true,
  "data": {
    "date": "2026-06-20",
    "overall_score": 4,
    "career": {"score": 4, "detail": "..."},
    "wealth": {"score": 3, "detail": "..."},
    "love": {"score": 5, "detail": "..."},
    "health": {"score": 3, "detail": "..."},
    "lucky_color": "红色",
    "lucky_number": "3, 8",
    "lucky_direction": "东方",
    "interpretation": "今日整体运势不错..."
  }
}
```

## 验收标准

```
Scenario: 今日运势已生成
  Given 用户已登录，今日运势已推送
  When  查看今日运势
  Then  应返回完整运势数据

Scenario: 今日运势未生成
  Given 用户已登录，今日运势尚未生成
  When  查看今日运势
  Then  应返回提示 "今日运势正在生成中，请稍后查看"
```

## 依赖项

- 前置：005-用户登录、018-每日推送调度
