# Spec 010: 八字排盘

> 模块：命理计算引擎 | 优先级：P0 | 预计：2 天

## 功能描述

根据出生年月日时，排算四柱八字、五行分析、十神关系、大运排列。

## 模块结构

```
fortune_engine/bazi/
├── calendar.py      # 万年历、阳历转阴历
├── pillar.py        # 天干地支排盘（年月日时四柱）
├── five_element.py  # 五行分析（旺衰、喜忌）
├── ten_god.py       # 十神关系计算
├── luck_cycle.py    # 大运、流年排盘
└── daily_fortune.py # 每日运势推算
```

## 输入/输出

**输入：**
```json
{
  "birth_year": 1990,
  "birth_month": 5,
  "birth_day": 15,
  "birth_hour": 8,
  "gender": 1,
  "birth_location": "北京"
}
```

**输出：**
```json
{
  "year_pillar": "庚午",
  "month_pillar": "辛巳",
  "day_pillar": "丙寅",
  "hour_pillar": "壬辰",
  "day_master": "丙",
  "five_elements": {"金": 2, "木": 1, "水": 1, "火": 2, "土": 2},
  "ten_gods": {...},
  "major_luck_cycles": [...],
  "favorable_elements": ["木", "火"]
}
```

## 验收标准

```
Scenario: 排盘结果准确
  Given 用户出生信息：1990-05-15 08:00 北京 男
  When  调用八字排盘
  Then  四柱结果应与万年历一致
  And   五行分布应正确
  And   十神关系应正确

Scenario: 子时处理正确
  Given 用户出生时间：23:30
  When  调用八字排盘
  Then  应正确识别为子时
  And   日柱应按子时规则处理

Scenario: 闰月处理正确
  Given 用户出生在农历闰四月
  When  调用八字排盘
  Then  应正确识别闰月
  And   排盘结果应准确
```

## 技术要点

- 使用 lunar-python 库进行阳历转阴历
- 节气换月（不是初一换月）
- 真太阳时校正（根据出生地经度）
- 子时（23:00-01:00）日柱切换规则
