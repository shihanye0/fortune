# Spec 024: 占卜历史与反馈

> 模块：占卜功能 | 优先级：P2 | 预计：0.5 天

## 功能描述

用户查看占卜历史记录，并对占卜结果进行反馈。

## API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/v1/divination/records | 占卜历史列表 |
| POST | /api/v1/divination/:id/feedback | 占卜反馈 |

## 验收标准

```
Scenario: 查看占卜历史
  Given 用户有 5 条占卜记录
  When  查看占卜历史
  Then  应按时间倒序显示
  And   每条显示时间、类型、问题摘要

Scenario: 对占卜结果反馈
  Given 占卜记录ID=1 存在
  When  用户提交评分和反馈
  Then  反馈应保存成功
```

## 依赖项

- 前置：022-六爻占卜交互、023-奇门遁甲交互
