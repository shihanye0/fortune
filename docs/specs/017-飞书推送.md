# Spec 017: 飞书机器人推送

> 模块：推送服务 | 优先级：P0 | 预计：0.5 天

## 功能描述

通过飞书自定义机器人 Webhook 推送消息卡片。

## 输入/输出

**输入：**
- 飞书 Webhook URL
- 运势数据

**输出：** 消息卡片发送成功/失败

## 验收标准

```
Scenario: 发送飞书消息成功
  Given 用户配置了有效的飞书 Webhook URL
  When  系统推送运势消息
  Then  飞书应收到消息卡片
  And   卡片应包含运势概要和详情链接

Scenario: Webhook URL 失效
  Given 用户配置的 Webhook URL 已失效
  When  推送失败
  Then  应重试 3 次
  And   都失败后通过 QQ 邮箱通知用户更新 Webhook
```

## 技术要点

- 飞书 Webhook：POST JSON 到 Webhook URL
- 消息卡片格式：使用飞书 Interactive Card
- 支持 Markdown 格式

- 外部：飞书机器人服务
