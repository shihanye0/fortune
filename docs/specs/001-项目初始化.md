# Spec 001: 项目初始化

> 模块：基础设施 | 优先级：P0 | 预计：0.5 天

## 功能描述

创建项目目录结构、初始化后端（FastAPI）和前端（Vue 3）项目、配置 Git 和 .gitignore。

## 输入/输出

**输入**：无（从零开始）

**输出**：
- 完整的项目目录结构
- 后端可启动：`uvicorn app.main:app`
- 前端可启动：`npm run dev`
- Git 仓库初始化完成
- `.gitignore` 配置完成

## 验收标准

```
Scenario: 后端项目可启动
  Given 项目初始化完成
  When  执行 uvicorn app.main:app --reload
  Then  服务应启动在 http://localhost:8000
  And   GET /health 应返回 {"status": "ok"}

Scenario: 前端项目可启动
  Given 项目初始化完成
  When  执行 npm run dev
  Then  服务应启动在 http://localhost:5173
  And   浏览器访问应显示首页

Scenario: .gitignore 生效
  Given 项目初始化完成
  When  查看 .gitignore
  Then  应包含 node_modules/、__pycache__/、.env、.internal/ 等排除规则
```

## 技术要点

- 后端：FastAPI + uvicorn + SQLAlchemy + Pydantic
- 前端：Vue 3 + Vite + TypeScript + Element Plus
- 使用 `npm create vite@latest` 初始化前端
- .gitignore 必须在第一次提交前配置

## 依赖项

- 无前置依赖
- 外部依赖：Python 3.11+、Node.js 18+
