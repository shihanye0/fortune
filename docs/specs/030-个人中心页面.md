# Spec 030: 个人中心页面

> 模块：前端页面 | 优先级：P1 | 预计：1 天

## 功能描述

个人中心页面，管理个人信息、推送设置、反馈历史。

## 页面结构

```
┌─────────────────────────────────┐
│  个人中心                       │
├─────────────────────────────────┤
│  基本信息                       │
│  用户名：张三                   │
│  邮箱：zhangsan@qq.com          │
│  [编辑]                         │
│                                 │
│  生辰信息                       │
│  出生：1990-05-15 08:00         │
│  八字：庚午 辛巳 丙寅 壬辰      │
│  [修改生辰]                     │
│                                 │
│  推送设置                       │
│  每日推送：[开/关]              │
│  推送渠道：[QQ邮箱/飞书/两者]   │
│  推送时间：[07:00]              │
│  飞书 Webhook：[____________]   │
│  [保存设置]                     │
│                                 │

│  [查看我的反馈]                 │
│                                 │
│  [注销账号]                     │
└─────────────────────────────────┘
```

## 验收标准

```
Scenario: 编辑推送设置
  Given 用户在个人中心
  When  修改推送时间为 "08:00" 并保存
  Then  设置应更新成功

Scenario: 修改生辰信息
  Given 用户在个人中心
  When  修改出生时辰并确认
  Then  八字应重新排算
  And   历史运势应保留
```
