# Spec 008: 账号注销

> 模块：用户管理 | 优先级：P2 | 预计：0.5 天

## 功能描述

用户注销账号，所有数据完全删除。

## API

```
DELETE /api/v1/users/me
```

**请求体：**
```json
{
  "password": "Abc123456"
}
```

## 验收标准

```
Scenario: 注销账号
  Given 用户已登录
  When  用户输入正确密码确认注销
  Then  用户数据应完全删除
  And   包括个人信息、八字档案、运势历史、占卜记录、反馈数据
  And   应返回成功消息

Scenario: 密码错误无法注销
  Given 用户已登录
  When  用户输入错误密码
  Then  应返回错误 "密码错误，无法注销"
  And   账号不应被删除
```

## 技术要点

- 需级联删除所有关联数据（bazi_profiles, daily_fortunes, divination_records）
- 必须验证密码才能注销
- 注销后 JWT Token 立即失效
