# Spec 016: QQ 邮箱推送

> 模块：推送服务 | 优先级：P0 | 预计：1 天

## 功能描述

通过 QQ 邮箱 SMTP 发送 HTML 格式的运势报告邮件。

## 输入/输出

**输入：**
- 收件人邮箱
- 运势数据（评分、维度、解读文字）

**输出：** HTML 格式邮件发送成功/失败

## 验收标准

```
Scenario: 发送运势邮件成功
  Given 用户邮箱 "zhangsan@qq.com" 有效
  When  系统发送运势报告邮件
  Then  邮件应成功发送
  And   邮件主题应为 "【命理运势】YYYY年MM月DD日 运势播报"
  And   邮件内容应为 HTML 格式，包含运势详情

Scenario: 邮件发送失败重试
  Given SMTP 服务器暂时不可用
  When  第一次发送失败
  Then  应重试，最多 3 次，间隔 5 分钟
  And   3 次都失败后应记录失败日志
```

## 技术要点

- QQ 邮箱 SMTP：smtp.qq.com:465 (SSL)
- 使用授权码登录（非密码）
- HTML 邮件模板需适配主流邮件客户端
- 附件大小限制：50MB

## 依赖项

- 前置：003-配置管理（SMTP 配置）
- 外部：QQ 邮箱 SMTP 服务
