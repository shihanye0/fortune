# Spec 009: 命理公共基础模块

> 模块：命理计算引擎 | 优先级：P0 | 预计：1 天

## 功能描述

定义天干、地支、五行、时辰等基础数据和关系，作为所有命理计算的基础。

## 模块结构

```
fortune_engine/common/
├── tiangan.py     # 天干定义（甲乙丙丁戊己庚辛壬癸）
├── dizhi.py       # 地支定义（子丑寅卯辰巳午未申酉戌亥）
├── wuxing.py      # 五行生克关系
└── shichen.py     # 时辰对照表
```

## 核心数据

| 类型 | 内容 |
|------|------|
| 天干 | 10 个天干、五行属性、阴阳属性 |
| 地支 | 12 个地支、五行属性、藏干 |
| 五行 | 生克关系（金木水火土相生相克） |
| 时辰 | 12 时辰与 24 小时对照 |
| 六十甲子 | 天干地支组合表 |

## 验收标准

```
Scenario: 天干五行属性正确
  Given 公共基础模块已实现
  When  查询天干 "甲" 的五行
  Then  应返回 "木"，阴阳为 "阳"

Scenario: 地支藏干正确
  Given 公共基础模块已实现
  When  查询地支 "子" 的藏干
  Then  应返回 "癸"

Scenario: 五行生克关系正确
  Given 公共基础模块已实现
  When  查询 "金" 生什么
  Then  应返回 "水"
  When  查询 "金" 克什么
  Then  应返回 "木"
```

## 技术要点

- 使用枚举或常量定义，确保类型安全
- 所有数据必须经过验证，确保准确性
- 编写完整的单元测试覆盖所有组合
