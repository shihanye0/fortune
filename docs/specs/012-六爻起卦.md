# Spec 012: 六爻起卦

> 模块：命理计算引擎 | 优先级：P1 | 预计：1 天

## 功能描述

实现六爻占卜的起卦算法（铜钱法、时间起卦），生成卦象数据。

## 模块结构

```
fortune_engine/liuyao/
├── hexagram.py    # 起卦算法
└── interpret.py   # 卦象基础解读规则
```

## 输入/输出

**输入：** 起卦方式（铜钱法/时间起卦）+ 用户问题

**输出：**
```json
{
  "question": "最近工作是否顺利",
  "method": "coin",
  "hexagram": {
    "name": "乾为天",
    "upper_trigram": "乾",
    "lower_trigram": "乾",
    "lines": [1, 1, 1, 1, 1, 1],
    "changing_lines": [2, 5],
    "changed_hexagram": {"name": "天火同人", ...}
  },
  "six_relatives": {...},
  "world_response": {...}
}
```

## 验收标准

```
Scenario: 铜钱法起卦
  Given 用户选择铜钱法起卦
  When  系统随机生成六爻
  Then  每爻应为老阴/少阴/少阳/老阳之一
  And   应生成本卦和变卦
  And   应标注六亲和世应

Scenario: 时间起卦
  Given 用户选择时间起卦
  When  系统根据当前时间起卦
  Then  卦象应基于当前年月日时计算
```

## 技术要点

- 铜钱法：随机生成 6 爻，每爻 3 次正反面
- 时间起卦：年月日时数字运算得出卦象
- 变卦：老阴变阳、老阳变阴
- 六亲：根据卦宫和五行关系确定
