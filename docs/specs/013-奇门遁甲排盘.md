# Spec 013: 奇门遁甲排盘

> 模块：命理计算引擎 | 优先级：P1 | 预计：2 天

## 功能描述

实现时家奇门遁甲排盘，计算九星、八门、八神落宫。

## 模块结构

```
fortune_engine/qimen/
├── chart.py       # 排盘算法
└── analysis.py    # 盘面分析规则
```

## 输入/输出

**输入：** 当前时辰（或指定时辰）

**输出：**
```json
{
  "time": "2026-06-20 08:00",
  "ju_shu": 3,
  "palace": {
    "1": {"star": "天蓬", "gate": "休门", "god": "值符", "element": "水"},
    "2": {"star": "天芮", "gate": "死门", "god": "螣蛇", "element": "土"},
    ...
    "9": {"star": "天英", "gate": "景门", "god": "太阴", "element": "火"}
  },
  "yin_yang": "阳遁",
  "dun_ju": 1
}
```

## 验收标准

```
Scenario: 排盘结果正确
  Given 当前时间 2026-06-20 08:00
  When  调用奇门遁甲排盘
  Then  九星落宫应正确
  And   八门落宫应正确
  And   八神落宫应正确
  And   阴阳遁和局数应正确
```

## 技术要点

- 时家奇门：以时辰为单位排盘
- 阳遁/阴遁：根据节气确定
- 局数：根据节气和日干支确定
- 九星、八门、八神的飞布规则
