# Spec 003: 配置管理

> 模块：基础设施 | 优先级：P0 | 预计：0.5 天

## 功能描述

统一管理项目配置，通过环境变量读取敏感信息，支持 .env 文件。

## 配置项

| 配置项 | 说明 | 示例 |
|--------|------|------|
| DATABASE_URL | 数据库连接 | mysql+pymysql://user:pass@localhost:3306/fortune |
| DEEPSEEK_API_KEY | DeepSeek API 密钥 | sk-xxx |
| DEEPSEEK_BASE_URL | DeepSeek API 地址 | https://api.deepseek.com/v1 |
| SMTP_HOST | QQ 邮箱 SMTP | smtp.qq.com |
| SMTP_PORT | SMTP 端口 | 465 |
| SMTP_USER | 发件邮箱 | your-qq@qq.com |
| SMTP_PASSWORD | SMTP 授权码 | xxx |
| JWT_SECRET | JWT 签名密钥 | random-secret-key |
| JWT_EXPIRE_HOURS | Token 过期时间 | 24 |
| APP_PORT | 服务端口 | 8000 |
| APP_ENV | 运行环境 | development/production |

## 验收标准

```
Scenario: 环境变量正常读取
  Given .env 文件配置正确
  When  启动应用
  Then  所有配置项应正确加载

Scenario: 缺少必填配置时报错
  Given DEEPSEEK_API_KEY 未配置
  When  启动应用
  Then  应报错提示缺少配置项
```

## 技术要点

- 使用 pydantic-settings 管理配置
- .env 文件不提交 Git，.env.example 提交
- 敏感配置启动时校验
