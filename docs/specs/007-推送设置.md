# Spec 007: 推送设置

> 模块：用户管理 | 优先级：P1 | 预计：0.5 天

## 功能描述

用户配置每日运势推送的渠道、时间和开关。

## API

```
PUT /api/v1/users/me/push-settings
```

**请求体：**
```json
{
  "push_enabled": true,
  "push_channel": "both",
  "push_time": "08:00",
  "feishu_webhook": "https://open.feishu.cn/open-apis/bot/v2/hook/xxx"
}
```

## 验收标准

```
Scenario: 开启每日推送
  Given 用户已登录
  When  用户设置 push_enabled=true, push_channel="email", push_time="07:00"
  Then  推送设置应保存成功
  And   从明天开始每天 07:00 推送

Scenario: 设置飞书推送但未填写 Webhook
  Given 用户已登录
  When  用户设置 push_channel="feishu" 但 feishu_webhook 为空
  Then  应返回错误 "飞书推送需要填写 Webhook URL"

Scenario: 关闭每日推送
  Given 用户已开启推送
  When  用户设置 push_enabled=false
  Then  系统应停止为该用户推送运势
```

## 技术要点

- push_time 格式为 HH:MM
- feishu_webhook 需验证 URL 格式
- 推送渠道：email / feishu / both
