# Spec 015: 用户反馈摘要

> 模块：LLM 解读服务 | 优先级：P1 | 预计：0.5 天

## 功能描述

汇总用户历史反馈，生成摘要，作为 LLM prompt 的一部分影响后续解读风格。

## 输入/输出

**输入：** 用户最近 30 条反馈数据

**输出：** 反馈摘要文本
```
用户偏好：
- 更关注事业运和财运
- 觉得财运分析很准，希望更详细
- 喜欢简洁风格，不要太长
- 过去反馈准确度评分平均 4.2/5
```

## 验收标准

```
Scenario: 生成反馈摘要
  Given 用户有 20 条反馈记录
  When  调用反馈摘要生成
  Then  应返回用户偏好的文字描述
  And   应包含关注维度、风格偏好、准确度评估

Scenario: 无反馈时返回默认
  Given 用户无反馈记录
  When  调用反馈摘要生成
  Then  应返回 "暂无用户反馈数据"
```

## 技术要点

- 只取最近 30 条反馈
- 统计评分标签分布
- 提取高频关键词
- 摘要长度控制在 200 字以内

## 依赖项

- 前置：004-用户注册（需要有反馈数据后才能工作）
