# Spec 023: 奇门遁甲交互

> 模块：占卜功能 | 优先级：P1 | 预计：1 天

## 功能描述

支持两种奇门遁甲使用模式：输入问题起盘、实时看盘。

## API

**模式一：输入问题起盘**
```
POST /api/v1/divination/qimen
{"question": "明天面试是否顺利", "mode": "question"}
```

**模式二：实时看盘**
```
POST /api/v1/divination/qimen
{"mode": "realtime"}
```

## 验收标准

```
Scenario: 输入问题起盘
  Given 用户已登录
  When  输入问题并选择奇门遁甲
  Then  应根据当前时辰排盘
  And   应调用 DeepSeek 解读盘面
  And   应给出针对性建议

Scenario: 实时看盘
  Given 用户已登录
  When  选择实时看盘模式
  Then  应显示当前时辰的奇门盘面
  And   应给出当前时辰的总体分析
```
