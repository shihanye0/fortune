# Spec 026: 注册页面

> 模块：前端页面 | 优先级：P0 | 预计：1 天

## 功能描述

用户注册页面，包含账号信息和生辰信息表单。

## 页面结构

```
注册表单：
├── 用户名（必填）
├── 邮箱（必填）
├── 密码（必填，≥8位，含字母和数字）
├── 确认密码
├── 出生日期（阳历，日期选择器）
├── 出生时辰（时间选择器，0-23时）
├── 性别（单选）
├── 出生地（选填，用于真太阳时校正）
└── [注册] 按钮
```

## 验收标准

```
Scenario: 注册成功
  Given 用户填写完整有效信息
  When  点击注册
  Then  应跳转到首页
  And   应自动登录

Scenario: 表单验证
  Given 用户在注册页面
  When  输入无效邮箱并提交
  Then  应显示邮箱格式错误提示
```
