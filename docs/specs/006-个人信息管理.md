# Spec 006: 个人信息管理

> 模块：用户管理 | 优先级：P1 | 预计：0.5 天

## 功能描述

用户查看和编辑个人信息、生辰信息。

## API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/v1/users/me | 获取个人信息 |
| PUT | /api/v1/users/me | 更新基本信息（用户名、手机号） |
| PUT | /api/v1/users/me/birth | 更新生辰信息 |

## 验收标准

```
Scenario: 查看个人信息
  Given 用户已登录
  When  调用 GET /api/v1/users/me
  Then  应返回用户信息（不含密码）
  And   应包含八字概要

Scenario: 修改生辰信息
  Given 用户已登录
  When  用户修改出生时辰从 "08:00" 改为 "09:00"
  Then  系统应重新排算八字
  And   历史运势应保留
  And   后续运势应按新八字推算
```

## 技术要点

- 修改生辰后需重新调用八字排盘引擎
- 历史运势保留，后续按新八字推算
- 返回数据脱敏（不返回密码哈希）

## 依赖项

- 前置：005-用户登录、010-八字排盘
