# Spec 022: 六爻占卜交互

> 模块：占卜功能 | 优先级：P1 | 预计：1 天

## 功能描述

用户输入问题进行六爻占卜，系统起卦并调用 LLM 解读。

## API

```
POST /api/v1/divination/liuyao
```

**请求体：**
```json
{
  "question": "最近工作是否顺利",
  "method": "coin"
}
```

**响应：**
```json
{
  "success": true,
  "data": {
    "id": 1,
    "question": "最近工作是否顺利",
    "hexagram": {...},
    "interpretation": "卦象显示..."
  }
}
```

## 验收标准

```
Scenario: 输入问题占卜
  Given 用户已登录
  When  输入问题并选择铜钱法起卦
  Then  应生成卦象
  And   应调用 DeepSeek 解读卦象
  And   应返回完整解读

Scenario: 不输入问题直接占卜
  Given 用户已登录
  When  不输入问题直接起卦
  Then  应允许起卦
  And   解读为 "总体运势" 分析
```

## 依赖项

- 前置：012-六爻起卦、014-DeepSeek解读服务
