# Spec 028: 每日运势页面

> 模块：前端页面 | 优先级：P0 | 预计：1 天

## 功能描述

展示今日运势和历史运势列表。

## 验收标准

```
Scenario: 查看今日运势
  Given 用户已登录，运势已生成
  When  进入运势页面
  Then  应显示今日运势卡片

Scenario: 查看历史列表
  Given 用户有历史运势
  When  向下滚动
  Then  应显示历史运势列表
```
