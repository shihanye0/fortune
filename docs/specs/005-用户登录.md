# Spec 005: 用户登录

> 模块：用户管理 | 优先级：P0 | 预计：0.5 天

## 功能描述

用户通过邮箱和密码登录，获取 JWT Token。

## API

```
POST /api/v1/auth/login
```

**请求体：**
```json
{
  "email": "zhangsan@qq.com",
  "password": "Abc123456"
}
```

**响应：**
```json
{
  "success": true,
  "data": {
    "token": "jwt-token...",
    "expires_in": 86400
  }
}
```

## 验收标准

```
Scenario: 使用正确凭据登录
  Given 用户已注册邮箱 "zhangsan@qq.com" 密码 "Abc123456"
  When  用户输入正确邮箱和密码
  Then  应返回 JWT Token

Scenario: 密码错误
  Given 用户已注册
  When  用户输入错误密码
  Then  应返回错误 "邮箱或密码错误"
  And   HTTP 状态码应为 401

Scenario: 邮箱不存在
  Given 邮箱未注册
  When  用户尝试登录
  Then  应返回错误 "邮箱或密码错误"（不暴露邮箱是否存在）
```

## 技术要点

- JWT Token 包含 user_id，过期时间 24 小时
- 登录失败不暴露邮箱是否存在（防枚举）
- 密码验证使用 bcrypt.checkpw
