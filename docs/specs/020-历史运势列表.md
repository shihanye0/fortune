# Spec 020: 历史运势列表

> 模块：运势查看 | 优先级：P1 | 预计：0.5 天

## 功能描述

用户查看历史运势列表，支持分页。

## API

```
GET /api/v1/fortunes?page=1&limit=20
```

**响应：**
```json
{
  "success": true,
  "data": [
    {"id": 1, "date": "2026-06-20", "overall_score": 4, "summary": "..."},
    {"id": 2, "date": "2026-06-19", "overall_score": 3, "summary": "..."}
  ],
  "meta": {"total": 30, "page": 1, "limit": 20, "total_pages": 2}
}
```

## 验收标准

```
Scenario: 查看历史运势
  Given 用户有 30 天运势记录
  When  查看运势列表
  Then  应按日期倒序显示
  And   每条显示日期、综合评分、摘要

Scenario: 无历史记录
  Given 用户刚注册，无运势记录
  When  查看运势列表
  Then  应返回空列表
```
